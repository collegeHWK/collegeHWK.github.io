ARPCBS

Augmented Reality Personal Computer Building Simulator

ARPCBS (Augmented Reality Personal Computer Building Simulator) is an application developed for the android platform. It serves to provide a streamlined service for consumers to obtain information about building their own custom PC.

The features includes a Step-By-Step guide for users who are new to PC-building or people who want to refer to basic installation steps. Other than that, this application provides a PC components and peripherals catalogue to allow users to learn about existing parts before making a purchase.
